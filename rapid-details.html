Building an OPC UA python client

In the summer of 2021, I worked with Rapid Effect, a company based in Singapore that deals with providing services for automation products. Specifically, I worked on creating a python client which would poll the value of nodes on a server and then log this information, and periodically send it to the cloud to be stored.

To test my client, I uploaded the code to a raspberry pi system, which would connect to a kepware server communicating using the OPC UA protocol. To add security to the client, I added certificates and authentication. The polled data was sent using an Azure API to the cloud, which I was able to test using the application 'Postman'. Finally, to make sure this client would would run on multiple different versions and operating systems, I was able to load and run the client on a docker image.

Flappy Bird Neural Network

  1. Home
  2. Flappy Bird Neural Network

A neural network trained using a genetic algorithm

In this imitation of the classic game 'Flappy Bird', a population of birds constantly try and survive for the longest time. Each bird controls itself individually using a neural network, using its position as well as the following pipe's position as input with the output value determining whether the bird should jump at any given frame. This game was created using Processing, an animation and GUI library built on top of Java. In the left, you can see the game running, while on the right, there is a representation of the bird in the current generation with the greatest fitness. Green lines indicate a positive weight while red lines indicate a negative weight. The thickness of the line indicates the strength of the weight.

When a bird collides with either the boundary of the screen or a pipe, its 'brain' in the form of a neural network is added to a mating pool a number of times. The number of times the bird's neural network is added is proportional to the fitness of the bird, determined by how long it has survived. When all birds in the current generation have collided, a new population is generated where each bird's brain is a random mix of two neural networks selected from the mating pool. Therefore those with the highest fitness have a greater chance of influencing the next generation. To make sure the birds have a chance to improve, each weight in the generated neural network has a small chance to mutate, changing its value slightly.
